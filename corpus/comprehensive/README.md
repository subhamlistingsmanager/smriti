# Comprehensive Epic Corpus

This folder contains full-story coverage datasets for the two major epics.

This is the current default runtime corpus structure for the app.

Generated files:

- episodes/ramayan-full.json
- episodes/mahabharat-full.json

Build command:

- npm run corpus:build:comprehensive

Notes:

- Full corpus is loaded by default in the app runtime.
- Corpus is prepared for multilingual retrieval (English/Hindi/Hinglish) with multilingual embeddings.
- Designed for local, voice-based questioning over complete Ramayana and Mahabharata story flow.
